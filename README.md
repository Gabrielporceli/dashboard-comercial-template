# Painel Comercial | Porceli Tracking

Dashboard para gestão e qualificação de leads comerciais com integração n8n e interface Liquid Glass.

## Tecnologias

- React
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Shadcn UI

## Funcionalidades

- Visualização de leads em tempo real via webhooks n8n
- Filtros avançados por plataforma e status
- Métricas consolidadas de conversão
- Interface premium com estética Liquid Glass
- Sincronização automática configurável

## Execução Local

1. Instale as dependências: `npm install`
2. Inicie o servidor: `npm run dev`
3. O painel estará disponível em `http://localhost:8080`
